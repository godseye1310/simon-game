Simon Memory Game

Press A Key to Start

Instruction

  • When the Game Begins it will flash a one Random box.
  • Player has to press the box that was flashed
  • If Correct next Random Box will be flashed, Player has to press the first box flashed(When game started) and the current box flashed in sequence
  • Each level the player has to press the whole previous sequence displayed and then the current box flashed
